    HyperText Markup Language (HTML)
    is the basic scripting language used by web browsers to render pages on the World Wide Web. HyperText allows a user to click a link and be redirected to a new page referenced by that link.
    Cascading Style Sheet (CSS)
    is a computer language for laying out and structuring webpages (HTML or XML). This language contains coding elements and is composed of these "cascading style sheets" which are equally called CSS files.
    JavaScript (JS)
    Is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.
